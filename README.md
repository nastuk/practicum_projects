## Описание проектов
Проекты были выполнены в рамках обучения на курсе "Специалист по Data Science" в Яндекс.Практикуме.

| Название проекта | Описание | Используемые библиотеки | 
| :---------------------- | :---------------------- | :---------------------- |
| [1. Исследование объявлений о продаже квартир в Санкт-Петербурге](https://github.com/nastuk/practicum_projects/tree/main/1.real_estate) | Проект анализирует данные о продаже квартир для определения рыночной стоимости объектов и выявления факторов, влияющих на цену. | *pandas, numpy, datetime, matplotlib, seaborn* |
| [2. Анализ сервиса аренды самокатов GoFast](https://github.com/nastuk/practicum_projects/tree/main/2.rides_stats) | Проект анализирует данные пользователей сервиса аренды для проверки гипотез, способствующих росту бизнеса (тариф, подписка, длительность поездок и их стоимость). | *pandas, numpy, datetime, matplotlib, seaborn, scipy, math* |
| [3. Исследование факторов, влияющих на коммерческий успех компьютерных игр](https://github.com/nastuk/practicum_projects/tree/main/3.games) | Проект анализирует ключевые факторы, определяющие популярность компьютерной игры и её потенциал для успешного продвижения на рынке в разных регионах. | *pandas, numpy, matplotlib, scipy, seaborn* |
| [4. Прогнозирование критериев отбора бурёнок в поголовье молочного хозяйства](https://github.com/nastuk/practicum_projects/tree/main/4.cows) | В проекте анализируются ключевые факторы, определяющие возможный удой коровы и вероятность получить вкусное молоко. | *pandas, numpy, matplotlib, seaborn, sklearn* |
| [5. Персонализированные предложения постоянным клиентам интернет-магазина «В один клик»](https://github.com/nastuk/practicum_projects/tree/main/5.customers) | Проект формулирует критерии для внедрения персонализированных предложений постоянным клиентам интернет-магазина, чтобы увеличить их покупательскую активность. | *pandas, numpy, matplotlib, seaborn, sklearn, scipy, phik, shap* |
| [6. Оптимизация бизнес-решений при управлении персоналом компании "Работа с заботой"](https://github.com/nastuk/practicum_projects/tree/main/6.job_satisfaction) | В проекте разрабатываются прогнозные модели, которые предсказывают уровень удовлетворённости сотрудника и вероятность его увольнения из компании. | *pandas, numpy, matplotlib, seaborn, sklearn, scipy, phik* |
| [7. Определение стоимости автомобилей](https://github.com/nastuk/practicum_projects/tree/main/7.car_price)| В проекте разработана модель для определения рыночной стоимости автомобиля на основе различных критериев (модель, год выпуска, пробег и др.). | *pandas, numpy, matplotlib, seaborn, sklearn* |
| [8. Выбор локации для скважины](https://github.com/nastuk/practicum_projects/tree/main/8.wells_location) | Проект анализирует критерии качества нефти и объём её запасов для определения региона, где добыча принесёт наибольшую прибыль. Возможная прибыль и риски проанализированы  техникой Bootstrap. | *pandas, numpy, matplotlib, seaborn, sklearn, phik, lightgbm* |
| [9. Прогнозирование заказов такси](https://github.com/nastuk/practicum_projects/tree/main/9.taxi_forecasting) | В проекте на основе исторических данных о заказах такси в аэропортах построена модель для предсказания количества заказов на следующий час. | *pandas, numpy, math, matplotlib, seaborn, statsmodels, lightgbm, catboost* |
| [10. Модерация токсичных комментариев для интернет-магазина «Викишоп»](https://github.com/nastuk/practicum_projects/tree/main/10.toxic_comments) | В проекте проводится сентимент-анализ и разработана модель для поиска токсичных комментариев. | *pandas, numpy, nltk, matplotlib, seaborn, string, re, collections, PIL, lightgbm* |
| [11. Определение возраста покупателей сетевого супермаркета «Хлеб-Соль»](https://github.com/nastuk/practicum_projects/tree/main/11.cv_age) | В проекте разрабатывается модель для определения приблизительного возраста покупателя супермаркета для контроля продажи алкоголя и разработки рекомендательных систем. | *pandas, numpy, keras, matplotlib, seaborn, tensorflow, PIL* |
| [12. Прогнозирование оттока клиентов из компании «ТелеДом»](https://github.com/nastuk/practicum_projects/tree/main/12.final_telecom) | В проекте анализируются различные особенности пользовательского опыта и разрабатывается модель для прогнозирования оттока клиентов, чтобы предлагать им промокоды и специальные условия. | *pandas, numpy, phik, matplotlib, seaborn, scipy, shap, lightgbm* |
